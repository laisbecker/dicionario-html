Dicionário de HTML

Tag de Imagem - img

A tag de imagem permite colocar na tela uma imagem local ou da internet.

Tag Negrito - strong

A tag strong é utilizada para destacar um texto importante, que ficará em negrito.

Tag Bold - b

A tag b é utilizada para deixar um texto em negrito, sem marcá-lo como importante.

Tag de Tabela - table

A tag table permite criar tabelas com linhas e colunas, utilizando os elementos abaixo.

Tag Link - Link

A tag link é utilizada para linkar uma folha de estilo externa.

Tag de Cabeçalho - Header

A tag header é utilizada para adicionar um cabeçalho ao artigo.

Tag de Divisão - Div

A tag div define uma divisão ou uma seção em um documento HTML. É utilizada como um contêiner para elementos HTML, que são então estilizados com CSS ou manipulados com JavaScript.
Além disso, pode ser facilmente estilizada usando o atributo class ou id, e qualquer tipo de conteúdo pode ser colocado dentro da tag div.
Observação: por padrão, os navegadores sempre colocam uma quebra de linha antes e depois do elemento div.

Tag de Pesquisa - Search

A tag search serve para cercar o formulário de pesquisa com um elemento de pesquisa.

Tag de Lista - Li

A tag li, cria um um item de uma lista ordenada (ol) ou uma lista não ordenada (ul).

Tag de Botão - Button

A tag button define um botão clicável
